Image QC Analyzer

This Flask-based Image QC application analyzes image attributes and text/font properties to ensure compliance with predefined standards. It extracts resolution, DPI, color profile, margins, and background type using Pillow, while OpenCV and pytesseract handle OCR-based text detection and font size estimation. The app compares extracted attributes against predefined client rulesets and highlights mismatches. A simple web interface allows users to upload images, view extracted data, and check compliance in real-time.

  • Python
  • Flask
  • OpenCV
  • pytesseract
  • HTML/CSS
  • JavaScript
